# AI Resume SaaS

A full-stack AI-powered Resume Builder with user authentication, resume history, credits system, and PDF downloads.

## Features

- Generate AI-powered resumes
- Save resume history for logged-in users
- Download resumes as PDF
- Credits system to limit monthly resume generation
- Responsive React frontend
- Node.js / Express backend

## Tech Stack

- **Frontend:** React, Axios, CSS
- **Backend:** Node.js, Express
- **Database:** MongoDB / PostgreSQL
- **PDF generation:** jsPDF
- **Deployment:** Render / Vercel / Railway
